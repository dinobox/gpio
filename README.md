# gpio

bash、shell下的操控GPIO的脚本。


## orangepi 的用法：

让脚本可执行

```
chmod +x ./power_orangepi.sh
```

让GPIO2点亮LED

```
./power_orangepi 2 1
```

关闭LED

```
./power_orangepi 2 0
```


## raspberrypi 的用法：

让脚本可执行：

```
chmod +x ./power_raspberrypi.sh
```

让GPIO2点亮LED

```
./power_raspberrypi 2 1
```

关闭LED

```
./power_raspberrypi 2 0
```
